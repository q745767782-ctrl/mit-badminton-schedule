# MIT Badminton Schedule

This is a dynamic rebuild of the old `shen.nz/badminton` page.

The site reads `schedule.json` at page load. The JSON is generated by:

```sh
npm install
npx playwright install chromium
npm run update
```

The updater opens MIT Recreation's Mazevo calendar link, waits for the current
seven-day schedule to render, and filters Open Rec badminton sessions. It keeps
shared Rockwell/du Pont open rec court slots as lower-confidence options unless
you run it with `INCLUDE_SHARED_SESSIONS=0`.

## Deployment

For GitHub Pages, the included workflow runs every 15 minutes and commits a
fresh `schedule.json`. The MIT Open Rec calendar code is public and already
included. If MIT changes the link, add or update a repository secret named
`MAZEVO_CALENDAR_CODE`.

`MAZEVO_API_KEY` is optional now. It is only used as a fallback if browser
rendering of the Mazevo calendar fails.

## Fallback

MIT Recreation also publishes `Open-Rec-Schedule.pdf`, but that file can lag far
behind the live Mazevo calendar, so it is intentionally not used by default.
