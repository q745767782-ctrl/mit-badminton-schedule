# MIT and BU Badminton Schedules

This is a dynamic rebuild of the old `shen.nz/badminton` page, with an
additional Boston University FitRec open recreation badminton page.

The MIT page reads `schedule.json` at page load. The BU page reads
`bu-schedule.json`. The JSON files are generated by:

```sh
npm install
npx playwright install chromium
npm run update:all
```

The MIT updater opens MIT Recreation's Mazevo calendar link, waits for the
current seven-day schedule to render, and filters Open Rec badminton sessions.
It keeps shared Rockwell/du Pont open rec court slots as lower-confidence
options unless you run it with `INCLUDE_SHARED_SESSIONS=0`.

The BU updater reads BU FitRec Hours, extracts the 3-Court Gym Hours section,
expands recurring Open Rec Badminton blocks across the next two weeks, and
removes closure dates that say badminton is unavailable.

## Deployment

For GitHub Pages, the included workflow runs four times per hour and commits
fresh `schedule.json` and `bu-schedule.json` files. The MIT Open Rec calendar
code is public and already included. If MIT changes the link, add or update a
repository secret named `MAZEVO_CALENDAR_CODE`.

`MAZEVO_API_KEY` is optional now. It is only used as a fallback if browser
rendering of the Mazevo calendar fails.

## Fallback

MIT Recreation also publishes `Open-Rec-Schedule.pdf`, but that file can lag far
behind the live Mazevo calendar, so it is intentionally not used by default.
